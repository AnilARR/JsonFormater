# JSON Formatter

🚀 **Live Demo:** https://anilarr.github.io/JsonFormater

A fast and lightweight JSON Formatter built with HTML, CSS, and JavaScript.

## Features

* ✅ Format JSON
* ✅ Minify JSON
* ✅ Validate JSON
* ✅ Copy JSON to Clipboard
* ✅ Error Detection
* ✅ Responsive Design
* ✅ No Backend Required
* ✅ Works on GitHub Pages

## Usage

1. Paste your JSON into the editor.
2. Click **Format JSON** to beautify the data.
3. Click **Minify JSON** to compress it.
4. Use **Copy** to copy the output.
5. Invalid JSON will display an error message.

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Local Development

Clone the repository:

```bash
git clone https://github.com/AnilARR/JsonFormater.git
```

Open `index.html` in your browser.

## Deploy on GitHub Pages

1. Push the repository to GitHub.
2. Open Repository Settings.
3. Go to Pages.
4. Select:

   * Branch: `main`
   * Folder: `/ (root)`
5. Save.


```text
https://anilarr.github.io/JsonFormater/
```
